## 二、项目概述

**项目简介**：滁州亭城GIS系统（TingChengGIS）是一套以AI辅助开发为核心的多模态空间信息系统，基于Spring Boot 3 + Leaflet + CesiumJS技术栈，实现了亭空间数据管理、电子地图展示、AI智能问答与多模型LLM集成、TSP路线规划与导航、千亭批量导入导出、VR/AR数据接口等核心功能。系统支持H2嵌入式与PostgreSQL+PostGIS双数据库，提供Docker部署、ZIP部署包、Windows EXE/MSI安装包三种发布方式，配备完整的CI/CD流水线。本项目面向滁州文旅数字化需求，属于空间信息与数字技术领域的典型应用。

#### UCP用例点估算

| 因子类型 | 因子名称 | 权重 | 取值 | 说明 |
| --- | --- | --- | --- | --- |
| 技术复杂度因子（TCF） | T1 分布式 | 2.0 | 1 | 单体架构 |
| | T2 响应时间 | 1.0 | 3 | ≤2s要求 |
| | T3 最终用户效率 | 1.0 | 3 | 地图交互响应 |
| | T4 内部处理复杂 | 1.0 | 3 | TSP求解 + 空间查询 |
| | T5 代码复用 | 1.0 | 4 | JPA/CesiumJS等 |
| | T6 易安装 | 0.5 | 4 | 一键部署脚本 |
| | T7 易使用 | 0.5 | 4 | 响应式UI |
| | T8 可移植 | 2.0 | 2 | Windows/Linux |
| | T9 易变更 | 1.0 | 3 | 模块化设计 |
| | T10 并发 | 1.0 | 2 | 低并发场景 |
| | T11 安全 | 1.0 | 3 | JWT认证 |
| | T12 第三方依赖 | 1.0 | 3 | AI API依赖 |
| | T13 培训 | 1.0 | 1 | 自用系统 |
| | **TCF = 0.6 + 0.01×Σ(权重×取值)** | | **0.6 + 0.01×35 = 0.95** | |
| 环境因子（EF） | E1 UML熟悉度 | 1.5 | 3 | |
| | E2 应用经验 | 0.5 | 2 | |
| | E3 面向对象经验 | 1.0 | 3 | |
| | E4 组长能力 | 0.5 | 4 | |
| | E5 团队动力 | 1.0 | 3 | |
| | E6 需求稳定性 | 2.0 | 2 | 中期有变更 |
| | E7 兼职人员 | -1.0 | 3 | |
| | E8 编程语言难度 | -1.0 | 2 | |
| | **EF = 1.4 + (-0.03)×Σ(权重×取值)** | | **1.4 + (-0.03)×12.5 = 1.4 - 0.375 = 1.025** | |
| | **调整后UCP = UUCP × TCF × EF** | | **UCP = 75 × 0.95 × 1.025 = 73.0** | |

> UCP法估算工作量 = 73.0 × 20（PHM生产率因子）= 1,460人时 ≈ 5人 × 6周 × 48h/周，与进度计划吻合。
