## 2. 核心代码模块说明

### 2.1 TSP路线规划算法（TspSolver.java）

如**图2.1**所示，TSP求解器使用2-opt局部搜索算法。

```
输入：N个亭子坐标 [(lat₁,lon₁), ... , (latₙ,lonₙ)]
  │
  ▼
Step 1: Nearest Neighbor 构建初始解
  │   从第一个点出发，每次选择最近的未访问点
  │   初始路径: [A₀, A₁, A₂, ..., Aₙ]
  │
  ▼
Step 2: 2-opt 迭代优化
  │   for i in 1..n-2:
  │     for j in i+1..n-1:
  │       交换边 (Aᵢ,Aᵢ₊₁) 和 (Aⱼ,Aⱼ₊₁)
  │       若总距离减少 → 接受交换
  │
  ▼
Step 3: 连续100次无改进 → 停止
  │
  ▼
输出：最优访问顺序 + 总距离
```
**图2.1 TSP求解器算法流程**：Nearest Neighbor构建初解→2-opt迭代优化→收敛停止。时间复杂度O(n²)。

```java
// TspSolver.java - 2-opt核心交换逻辑
public RouteResult solve(List<Point> points, String objective) {
    int n = points.size();
    if (n < 2) return null;
    
    // Step 1: Nearest Neighbor 初始化
    int[] route = nearestNeighbor(points);
    double bestDist = calcTotalDistance(route, points);
    
    // Step 2: 2-opt 迭代
    boolean improved = true;
    int noImproveCount = 0;
    while (noImproveCount < MAX_NO_IMPROVE) {
        improved = false;
        for (int i = 1; i < n - 1; i++) {
            for (int j = i + 1; j < n; j++) {
                double delta = calcSwapDelta(route, points, i, j);
                if (delta < -1e-10) {  // 有改进
                    twoOptSwap(route, i, j);
                    bestDist += delta;
                    improved = true;
                }
            }
        }
        noImproveCount = improved ? 0 : noImproveCount + 1;
    }
    return new RouteResult(route, bestDist);
}
```

### 2.2 Haversine空间距离计算（GeoUtils.java）

如**图2.2**所示，系统使用Haversine公式计算球面距离，精度误差<0.5%。

```java
// GeoUtils.java - Haversine球面距离
public static double haversine(double lat1, double lon1,
                                double lat2, double lon2) {
    double dLat = Math.toRadians(lat2 - lat1);
    double dLon = Math.toRadians(lon2 - lon1);
    double a = Math.sin(dLat/2) * Math.sin(dLat/2)
             + Math.cos(Math.toRadians(lat1))
             * Math.cos(Math.toRadians(lat2))
             * Math.sin(dLon/2) * Math.sin(dLon/2);
    double c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a));
    return EARTH_RADIUS_KM * c;  // 6371 km
}
```
**图2.2 Haversine距离公式实现**：计算两点间球面大圆距离，用于空间查询与TSP距离矩阵生成。

```
  │
  ▼
Step 2: 绘制底图（坐标网格+标注）
  │
  ▼
Step 3: 逐帧添加路线段
  │   帧1: A→B, 帧2: A→B→C, ..., 帧N: 完整路径
  │
  ▼

  │   GifSequenceWriter 写入各帧
  │
  ▼
输出: route_animation.gif → System.getProperty("user.dir")
```
**图2.4 GIF导出流程**：Java 2D逐帧渲染后经GIF编码器输出为动态路线动画。
