### 1.2 核心用例描述

#### UC01：浏览地图

| 项目 | 内容 |
| --- | --- |
| **用例名称** | 浏览地图 |
| **参与者** | 所有用户 |
| **前置条件** | 打开浏览器访问系统 |
| **基本流程** | 1. 系统加载Leaflet 2D地图底图<br>2. 加载亭子标记图层（含聚合效果）<br>3. 用户平移/缩放地图浏览<br>4. 可选切换至CesiumJS 3D地球模式<br>5. 点击标记弹出亭子信息卡片 |
| **后置条件** | 无 |
| **涉及GIS技术** | Leaflet渲染、标记聚合、WMS/XYZ底图、CesiumJS 3D |

#### UC02：查询景点

| 项目 | 内容 |
| --- | --- |
| **用例名称** | 查询景点 |
| **参与者** | 所有用户 |
| **前置条件** | 地图已加载 |
| **基本流程** | 1. 用户输入关键词或选择类型筛选<br>2. 系统调用GET /pavilions/search?keyword=xx<br>3. 返回匹配亭子列表<br>4. 地图自动定位到首个结果并高亮标记 |
| **后置条件** | 无 |
| **涉及GIS技术** | 空间模糊查询、坐标定位 |

#### UC03：AI智能问答

| 项目 | 内容 |
| --- | --- |
| **用例名称** | AI智能问答 |
| **参与者** | 所有用户 |
| **前置条件** | 无 |
| **基本流程** | 1. 用户在AI对话窗口输入问题（文本或语音）<br>2. 系统调用AiService.chat()<br>3. AiService根据配置选择LLM提供商（OpenAI/DeepSeek/智谱）<br>4. LLM返回回答内容<br>5. 对话显示在界面中（流式输出） |
| **后置条件** | 对话记录保存在当前会话 |
| **异常流程** | LLM API不可用时，返回本地预置知识库内容 |
| **涉及AI技术** | 多模型LLM轮询、流式输出、语义理解 |

#### UC04：TSP路线规划

| 项目 | 内容 |
| --- | --- |
| **用例名称** | TSP路线规划 |
| **参与者** | 注册用户 |
| **前置条件** | 已登录，已选中≥2个景点 |
| **基本流程** | 1. 用户在地图选取N个亭子<br>2. 选择优化目标（最短距离/最少时间/最少费用）<br>3. 选择交通方式（驾车/骑行/步行）<br>4. 系统调用TspSolver.solve()以2-opt算法求解<br>5. 在地图上以逐段高亮动画绘制最优路径<br>6. 显示总距离、总时长与访问顺序 |
| **后置条件** | 路线可保存为RoutePlan，可选GIF导出 |
| **涉及多模态技术** | TSP优化算法、Haversine距离计算、逐段动画渲染、GIF编码 |

#### UC05：管理景点（CRUD）
